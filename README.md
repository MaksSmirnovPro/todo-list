# ✅ To Do List

Современное веб-приложение для управления задачами с приоритетами и датами.

## 🌟 Особенности

- 📝 Создание задач с текстом, приоритетом и датой
- 🎯 Фильтрация по статусу (все/активные/выполненные)
- 📊 Сортировка по алфавиту, приоритету и дате
- 📅 Отображение даты создания задачи
- 💾 Автосохранение в localStorage
- 📱 Адаптивный дизайн
- ⚛️ Построено на React

## 🚀 Демо

[Открыть приложение](https://MaksSmirnovPro.github.io/todo-list)

## 🛠️ Технологии

- React 18
- CSS3 (Flexbox, Grid, Анимации)
- LocalStorage API
- GitHub Pages

## 📦 Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/MaksSmirnovPro/todo-list.git

# Установка зависимостей
cd todo-list
npm install

# Запуск в режиме разработки
npm start

# Сборка для production
npm run build
